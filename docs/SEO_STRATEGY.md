# SEO Стратегия для qirim.online и ana-yurt.com

## Анализ текущего состояния

### qirim.online
- **Тип:** Музыкальный архив (SPA на React)
- **Плюсы:** Schema.org разметка, мультиязычность, аналитика
- **Минусы:** SPA плохо индексируется поисковиками, требует JS

### ana-yurt.com
- **Тип:** Культурно-образовательный портал
- **Плюсы:** 282 страницы контента, словари, видео
- **Минусы:** Нужно проверить мета-теги, sitemap

---

## Стратегия для qirim.online

### 1. Техническое SEO (Приоритет: Высокий)

#### 1.1 Server-Side Rendering (SSR) или Prerendering
Проблема: SPA на React плохо индексируется. Решения:
- **Prerender.io** - сервис для пререндеринга страниц для ботов
- **react-snap** - статическая генерация при билде
- **Next.js** - переход на SSR (долгосрочно)

#### 1.2 Sitemap.xml
Создать динамический sitemap со всеми:
- Альбомами
- Артистами
- Плейлистами
- Статическими страницами

#### 1.3 Robots.txt
```
User-agent: *
Allow: /
Sitemap: https://qirim.online/sitemap.xml

# Закрыть служебные разделы
Disallow: /api/
Disallow: /rest/
Disallow: /auth/
```

#### 1.4 Мета-теги для страниц
Динамические мета-теги для каждой страницы:
- Альбом: "Альбом {название} - {артист} | Qirim.Online"
- Артист: "{Имя артиста} - все песни | Qirim.Online"

### 2. Контентная стратегия

#### 2.1 Статические страницы (новые)
- `/about` - О проекте, история создания
- `/artists` - Каталог артистов с биографиями
- `/genres` - Описание жанров крымскотатарской музыки
- `/blog` - Статьи о музыке, культуре

#### 2.2 Уникальный контент
- Биографии артистов
- История песен
- Тексты песен (lyrics) - ОЧЕНЬ важно для SEO!
- Переводы песен

### 3. Ссылочная стратегия

#### 3.1 Внутренняя перелинковка
- Связать с ana-yurt.com (взаимные ссылки)
- Ссылки на артистов из альбомов и наоборот

#### 3.2 Внешние ссылки (белые методы)
- Регистрация в каталогах крымскотатарских ресурсов
- Упоминания в Wikipedia (статьи о крымских татарах)
- Пресс-релизы в крымскотатарских СМИ
- Партнёрство с культурными организациями

### 4. Локальное SEO
- Google Business Profile (если применимо)
- Яндекс.Справочник

## Стратегия для ana-yurt.com

### 1. Техническое SEO

#### 1.1 Проверить и исправить
- [ ] Sitemap.xml - наличие и актуальность
- [ ] Robots.txt - правильная конфигурация
- [ ] Canonical URLs - избежать дублей
- [ ] Скорость загрузки (PageSpeed)
- [ ] Mobile-friendly

#### 1.2 Структурированные данные
Добавить Schema.org разметку:
- Article - для статей
- VideoObject - для видео
- FAQPage - для словаря
- BreadcrumbList - хлебные крошки

### 2. Контентная стратегия

#### 2.1 Существующий контент
- Оптимизировать заголовки H1, H2, H3
- Добавить мета-описания к каждой странице
- Улучшить alt-теги изображений

#### 2.2 Новый контент
- Регулярные статьи (минимум 2-4 в месяц)
- Видеоуроки крымскотатарского языка
- Интерактивные тесты

### 3. Социальные сигналы
- Telegram канал
- YouTube канал
- Facebook страница
- Instagram
